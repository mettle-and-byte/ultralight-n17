# MnB Ultralight N17

The Ultralight N17 is a CAN-FD smart motor driver designed as a compact "backpack" that mounts directly on NEMA17 stepper motors. It features the TMC2240 driver with support for up to 3A continuous current and 36V maximum voltage.

## Key Features

- **CAN-FD Communication** – High-speed, robust networking compatible with RepRapFirmware (RRF), Klipper, and custom firmware
- **TMC2240 Stepper Driver** – Up to 3A continuous, 36V max
- **RP2350A Microcontroller** – Powerful processing with firmware customization support
- **Encoder Support** – Compatible with encoders for closed-loop operation
- **NEMA17 Backpack Design** – Mounts directly on motors, minimizing wiring complexity
- **Built-in Braking Resistor** – 2W resistor for overvoltage protection
